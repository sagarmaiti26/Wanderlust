Project: Wanderlust - House Rental Website

LIVE: https://wanderlust-0yl3.onrender.com/listings

Description:

Developed a full-stack property rental website using MongoDB, Express.js, Node.js, Bootstrap and followed RESTful principles in designing the API for communication between the frontend and backend. Wanderlust enables users to create accounts, add property listings, view them on an interactive map, and leave reviews. The project draws inspiration from popular platforms like Airbnb, emphasizing a seamless user experience in the property rental domain.

Key Contributions:

User Authentication:
Implemented secure user authentication and account creation functionalities, ensuring a personalized experience for users.

Listing Management:
Designed a user-friendly interface for property owners to add, edit, and manage their listings, including details such as property type, location, and pricing.

Interactive Map Integration:
Utilized mapping technologies to display property locations dynamically, providing users with a visual representation of available rental options.

Review System: 
Developed a robust review system, allowing users to share feedback and ratings on the listed properties, fostering trust and transparency within the community.

Responsive Design: 
Utilized Bootstrap to create a responsive and mobile-friendly layout, ensuring a consistent experience across various devices.

Technologies Used:

Frontend: Bootstrap for responsive design.
Backend: Node.js and Express.js for server-side logic.
Database: MongoDB for efficient data storage and retrieval.
Outcome:
Wanderlust successfully combines functionality and aesthetics, creating a platform that facilitates property rental transactions and enhances the overall user experience. The project demonstrates proficiency in full-stack web development and highlights skills in user authentication, database management, and map integration.

Future Enhancements:
Plan to implement additional features such as real-time messaging, advanced search filters, and payment integration to further enhance the platform's capabilities.
